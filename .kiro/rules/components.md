# Component Library

## Available Components

- Navigation bar
- Hero section
- Project cards
- Contact form
- Footer

## CSS Classes

- `.container` - Main content wrapper
- `.btn` - Button styles
- `.card` - Card component
- `.grid` - Grid layout
- `.flex` - Flexbox utilities

## Responsive Breakpoints

- Mobile: < 768px
- Tablet: 768px - 1024px
- Desktop: > 1024px
