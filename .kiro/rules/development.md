# Website Development Rules

## Project Structure

- Static HTML/CSS/JS website
- CSS organized in components, main, and responsive files
- Images stored in categorized folders (hero, icons, projects)
- Single page application structure

## Development Preferences

- Use semantic HTML5 elements
- Mobile-first responsive design
- Vanilla JavaScript (no frameworks)
- CSS Grid and Flexbox for layouts
- Optimize images for web
- Keep code minimal and clean

## File Organization

- CSS: Use BEM methodology for class naming
- JS: Use ES6+ features, modular approach
- Images: WebP format preferred, include alt text
- HTML: Proper semantic structure, accessibility compliant

## Performance

- Minimize HTTP requests
- Compress images
- Use efficient CSS selectors
- Avoid inline styles and scripts
