# プロダクト概要

個人ポートフォリオサイト - Web開発者向けのモダンなポートフォリオWebサイト

## 目的

開発者が自身のプロジェクト、スキル、経歴を紹介するためのレスポンシブなポートフォリオサイト。

## 主な機能

- プロフィール表示（名前、役職、自己紹介、SNSリンク）
- プロジェクト一覧表示（カテゴリ別：Webアプリ、プログラム、Chrome拡張機能）
- レスポンシブデザイン（モバイル、タブレット、デスクトップ対応）
- アクセシビリティ対応（WCAG 2.1 AA準拠）
- GitHub Pages対応

## ターゲットユーザー

- Web開発者
- フロントエンドエンジニア
- 技術系の求職者

## コンテンツ管理

- `src/data/config.json` - 個人情報とプロジェクトデータを一元管理
- JSONファイルを編集するだけでサイト内容を更新可能
